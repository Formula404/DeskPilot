# DeskPilot 接口与事件协议

## 1. 目标

本文档定义 Tauri 前端、浏览器扩展和 FastAPI 后端之间的第一版接口协议。

## 2. 基础约定

- 协议：HTTP + SSE + WebSocket。
- Tauri 前端使用 HTTP 提交任务，使用 SSE 接收任务事件。
- 浏览器扩展使用 WebSocket 作为主通道，接收后端下发的网页采集和页面动作指令；HTTP 仅作为兼容/兜底。
- 地址：`http://127.0.0.1:8765`。
- 数据格式：JSON。
- 时间格式：ISO 8601。
- ID 格式：UUID。

## 3. API 列表

### POST /chat

提交用户任务。

请求：

```json
{
  "message": "总结当前网页并保存",
  "source": "floating_window",
  "context_id": "optional-context-id"
}
```

响应：

```json
{
  "task_id": "uuid",
  "status": "queued"
}
```

### GET /tasks/{task_id}

查询任务状态。

响应：

```json
{
  "task_id": "uuid",
  "status": "running",
  "intent": "web_page_summary",
  "created_at": "2026-06-25T00:00:00Z",
  "updated_at": "2026-06-25T00:00:01Z",
  "result": null,
  "error": null
}
```

### POST /tasks/{task_id}/cancel

取消任务。

响应：

```json
{
  "ok": true
}
```

### GET /events

SSE 事件流。

事件类型：

- `task.created`
- `task.started`
- `task.plan.updated`
- `tool.started`
- `tool.finished`
- `tool.call.requested`
- `tool.observation.created`
- `approval.required`
- `task.completed`
- `task.failed`
- `task.cancelled`

事件格式：

```json
{
  "event_id": "uuid",
  "task_id": "uuid",
  "type": "tool.started",
  "message": "正在读取当前网页",
  "payload": {},
  "created_at": "2026-06-25T00:00:00Z"
}
```

### GET /context/current-window

获取当前 Windows 前台窗口。

响应：

```json
{
  "window_id": "string",
  "process_name": "chrome.exe",
  "process_path": "C:/Program Files/Google/Chrome/Application/chrome.exe",
  "title": "Page title",
  "rect": {
    "x": 0,
    "y": 0,
    "width": 1280,
    "height": 720
  }
}
```

### POST /context/browser/current-page

兼容接口。由浏览器扩展或调试工具上报当前页面上下文。

产品主链路不要求用户点击扩展按钮触发该接口；主链路应由后端通过浏览器 WebSocket 通道主动请求扩展采集当前页，后端收到结果后再写入同一张 `browser_contexts` 表。

请求：

```json
{
  "tab_id": 1,
  "url": "https://example.com",
  "title": "Example",
  "visible_text": "page text",
  "dom_summary": [],
  "captured_at": "2026-06-25T00:00:00Z"
}
```

### WebSocket /browser/ws

浏览器扩展连接本地服务的主通道。用于后端主动请求当前页上下文、下发受控页面动作，并接收扩展执行结果。

连接地址：

```text
ws://127.0.0.1:8765/browser/ws
```

扩展注册：

```json
{
  "type": "browser.register",
  "client_id": "uuid",
  "browser": "chrome",
  "capabilities": ["collect_page", "click", "type", "scroll", "extract_table"],
  "active_tab": {
    "tab_id": 1,
    "url": "https://example.com",
    "title": "Example"
  }
}
```

后端下发采集当前页：

```json
{
  "type": "browser.command",
  "request_id": "uuid",
  "command": "collect_page",
  "target": {
    "tab": "active"
  },
  "payload": {
    "include_visible_text": true,
    "include_dom_summary": true,
    "max_text_chars": 30000
  }
}
```

扩展返回结果：

```json
{
  "type": "browser.result",
  "request_id": "uuid",
  "ok": true,
  "command": "collect_page",
  "data": {
    "tab_id": 1,
    "url": "https://example.com",
    "title": "Example",
    "visible_text": "page text",
    "dom_summary": [],
    "captured_at": "2026-06-25T00:00:00Z"
  },
  "error": null
}
```

后端下发页面动作：

```json
{
  "type": "browser.command",
  "request_id": "uuid",
  "command": "click",
  "target": {
    "tab": "active"
  },
  "payload": {
    "selector": "button[type='submit']"
  }
}
```

错误结果：

```json
{
  "type": "browser.result",
  "request_id": "uuid",
  "ok": false,
  "command": "click",
  "data": null,
  "error": {
    "code": "ELEMENT_NOT_FOUND",
    "message": "未找到目标元素",
    "detail": {
      "selector": "button[type='submit']"
    }
  }
}
```

响应：

```json
{
  "context_id": "uuid",
  "ok": true
}
```

### GET /tools

返回已注册工具。该接口用于调试和设置页展示；Agent 内部还需要把工具转换为 OpenAI tool calling schema，并按 intent 做白名单过滤。

响应：

```json
{
  "tools": [
    {
      "name": "browser.get_current_page",
      "openai_name": "browser_get_current_page",
      "risk_level": "low",
      "description": "获取当前网页上下文",
      "input_schema": {
        "type": "object",
        "properties": {}
      }
    }
  ]
}
```

### POST /approvals/{approval_id}

提交审批结果。

请求：

```json
{
  "approved": true,
  "reason": ""
}
```

响应：

```json
{
  "ok": true
}
```

## 4. 任务状态

```text
queued
running
waiting_approval
completed
failed
cancelled
```

## 5. 工具结果协议

Tool Registry 内部工具结果：

```json
{
  "ok": true,
  "data": {},
  "message": "",
  "artifacts": [
    {
      "type": "file",
      "path": "data/exports/summary.md"
    }
  ],
  "error": null
}
```

OpenAI tool calling observation 写回模型时，必须使用精简 JSON，不能无控制地塞入完整网页正文或大文件内容。

## 6. 错误协议

```json
{
  "ok": false,
  "error": {
    "code": "BROWSER_CONTEXT_NOT_FOUND",
    "message": "未获取到当前网页上下文",
    "detail": {}
  }
}
```

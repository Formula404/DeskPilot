# DeskPilot 接口与事件协议

## 1. 目标

本文档定义 Tauri 前端、浏览器扩展和 FastAPI 后端之间的第一版接口协议。

## 2. 基础约定

- 协议：HTTP + SSE。
- 地址：`http://127.0.0.1:8765`。
- 数据格式：JSON。
- 时间格式：ISO 8601。
- ID 格式：UUID。

## 3. API 列表

### POST /chat

提交用户任务。

请求：

```json
{
  "message": "总结当前网页并保存",
  "source": "floating_window",
  "context_id": "optional-context-id"
}
```

响应：

```json
{
  "task_id": "uuid",
  "status": "queued"
}
```

### GET /tasks/{task_id}

查询任务状态。

响应：

```json
{
  "task_id": "uuid",
  "status": "running",
  "intent": "web_page_summary",
  "created_at": "2026-06-25T00:00:00Z",
  "updated_at": "2026-06-25T00:00:01Z",
  "result": null,
  "error": null
}
```

### POST /tasks/{task_id}/cancel

取消任务。

响应：

```json
{
  "ok": true
}
```

### GET /events

SSE 事件流。

事件类型：

- `task.created`
- `task.started`
- `task.plan.updated`
- `tool.started`
- `tool.finished`
- `approval.required`
- `task.completed`
- `task.failed`
- `task.cancelled`

事件格式：

```json
{
  "event_id": "uuid",
  "task_id": "uuid",
  "type": "tool.started",
  "message": "正在读取当前网页",
  "payload": {},
  "created_at": "2026-06-25T00:00:00Z"
}
```

### GET /context/current-window

获取当前 Windows 前台窗口。

响应：

```json
{
  "window_id": "string",
  "process_name": "chrome.exe",
  "process_path": "C:/Program Files/Google/Chrome/Application/chrome.exe",
  "title": "Page title",
  "rect": {
    "x": 0,
    "y": 0,
    "width": 1280,
    "height": 720
  }
}
```

### POST /context/browser/current-page

由浏览器扩展上报当前页面上下文。

请求：

```json
{
  "tab_id": 1,
  "url": "https://example.com",
  "title": "Example",
  "visible_text": "page text",
  "dom_summary": [],
  "captured_at": "2026-06-25T00:00:00Z"
}
```

响应：

```json
{
  "context_id": "uuid",
  "ok": true
}
```

### GET /tools

返回已注册工具。

响应：

```json
{
  "tools": [
    {
      "name": "browser.get_current_page",
      "risk_level": "low",
      "description": "获取当前网页上下文"
    }
  ]
}
```

### POST /approvals/{approval_id}

提交审批结果。

请求：

```json
{
  "approved": true,
  "reason": ""
}
```

响应：

```json
{
  "ok": true
}
```


## 5. 工具结果协议

```json
{
  "ok": true,
  "data": {},
  "message": "",
  "artifacts": [
    {
      "type": "file",
      "path": "data/exports/summary.md"
    }
  ],
  "error": null
}
```

## 6. 错误协议

```json
{
  "ok": false,
  "error": {
    "code": "BROWSER_CONTEXT_NOT_FOUND",
    "message": "未获取到当前网页上下文",
    "detail": {}
  }
}
```

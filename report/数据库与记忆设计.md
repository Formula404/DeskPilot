# DeskPilot 数据库与记忆设计

## 1. 目标

第一版统一使用 SQLite 保存配置、任务、审批、日志和长期记忆。长期记忆检索使用 SQLite FTS5，不引入向量数据库。

## 2. 数据库文件

```text
data/deskpilot.sqlite3
```

## 3. 表设计

### settings

保存用户配置。

```text
key TEXT PRIMARY KEY
value TEXT NOT NULL
updated_at TEXT NOT NULL
```

### task_runs

保存任务主记录。

```text
id TEXT PRIMARY KEY
user_message TEXT NOT NULL
intent TEXT
status TEXT NOT NULL
risk_level TEXT
result_summary TEXT
error_code TEXT
error_message TEXT
created_at TEXT NOT NULL
updated_at TEXT NOT NULL
```

### task_steps

保存任务执行步骤。

```text
id TEXT PRIMARY KEY
task_id TEXT NOT NULL
step_index INTEGER NOT NULL
type TEXT NOT NULL
name TEXT NOT NULL
input_json TEXT
output_json TEXT
status TEXT NOT NULL
created_at TEXT NOT NULL
finished_at TEXT
```

### approvals

保存敏感操作审批记录。

```text
id TEXT PRIMARY KEY
task_id TEXT NOT NULL
risk_level TEXT NOT NULL
summary TEXT NOT NULL
payload_json TEXT
status TEXT NOT NULL
created_at TEXT NOT NULL
resolved_at TEXT
```

### browser_contexts

保存浏览器扩展上报的当前页上下文。

```text
id TEXT PRIMARY KEY
tab_id TEXT
url TEXT NOT NULL
title TEXT
visible_text TEXT
dom_summary_json TEXT
captured_at TEXT NOT NULL
created_at TEXT NOT NULL
```

### app_profiles

保存应用路径、窗口匹配规则、RPA 配置。

```text
id TEXT PRIMARY KEY
app_key TEXT NOT NULL
display_name TEXT NOT NULL
process_name TEXT
exe_path TEXT
window_title_pattern TEXT
rpa_config_json TEXT
created_at TEXT NOT NULL
updated_at TEXT NOT NULL
```

### user_preferences

保存结构化偏好。

```text
key TEXT PRIMARY KEY
value TEXT NOT NULL
source TEXT NOT NULL
updated_at TEXT NOT NULL
```

### memory_items

保存长期记忆。

```text
id TEXT PRIMARY KEY
kind TEXT NOT NULL
content TEXT NOT NULL
source_task_id TEXT
sensitivity TEXT NOT NULL
created_at TEXT NOT NULL
updated_at TEXT NOT NULL
```

### memory_fts

FTS5 虚拟表。

```text
content
kind
```

## 4. 记忆写入规则

自动写入：

- 任务记录。
- 工具调用摘要。
- 应用路径。
- 用户明确设置的偏好。

默认不写入长期记忆：

- 微信聊天正文。
- 截图 OCR 原文。
- 用户输入的敏感账号、密码、令牌。
- 浏览器页面全文。

可写入长期记忆：

- 用户明确说“记住”。
- 成功的 RPA 定位策略。
- 常用导出目录。
- 常用应用路径。

## 5. 记忆读取规则

每个任务最多读取：

- 5 条结构化偏好。
- 5 条 FTS5 长期记忆。
- 当前任务相关的最近 3 条任务摘要。

读取结果写入 Agent state 的 `memory_refs` 字段。

## 6. 数据保留规则

第一版默认：

- 任务日志保留 30 天。
- 截图保留 7 天。
- 导出文件永久保留，除非用户删除。
- 审批记录永久保留。
